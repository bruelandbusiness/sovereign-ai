## Repurposing Strategy

Each day's content is written as a single core idea adapted across platforms. To maximize efficiency:

1. **Write the LinkedIn post first** - it's the longest and most detailed
2. **Adapt for Facebook** - shorten, make conversational, add a question
3. **Adapt for Instagram** - visual-first, create carousel or Reel script from the key points
4. **Adapt for X/Twitter** - distill to the sharpest take, consider threading for depth

**Batch creation recommendation:** Write all 30 days of content in 2-3 sessions. Schedule using a tool like Buffer, Hootsuite, or Later. Spend 15-20 minutes per day on live engagement (responding to comments, answering questions).
