## Campaign 3: Display - Retargeting

### Campaign Settings

| Setting | Value |
|---|---|
| **Campaign Name** | `SovAI - Display - Retargeting` |
| **Campaign Objective** | Leads (return visitors, form completions) |
| **Campaign Type** | Display |
| **Networks** | Google Display Network |
| **Monthly Budget** | $2,000 - $3,000 ($66 - $100/day) |
| **Bid Strategy** | Target CPA ($45) |
| **Expected CPC** | $0.50 - $2.50 |
| **Expected Conversion Rate** | 1.5% - 3% |
| **Target CPA** | $30 - $55 |
| **Expected Monthly Conversions** | 35 - 100 |

### Audience Configuration

**Primary Audience: Website Visitors - Non-Converters**

- **Source:** Google Ads remarketing tag (must be installed on all trysovereignai.com pages)
- **Include:** All visitors to trysovereignai.com in the last 30 days
- **Exclude:** Users who completed a conversion (form submission, demo booking, audit request)
- **Exclude:** Users who visited fewer than 2 pages (bounce filter)
- **Membership Duration:** 30 days

**Secondary Audience: High-Intent Non-Converters**

- **Include:** Visitors who viewed pricing page OR features page but did not convert
- **Membership Duration:** 14 days
- **Bid Adjustment:** +25%

**Tertiary Audience: Cart/Form Abandoners**

- **Include:** Visitors who started the free audit form but did not submit
- **Membership Duration:** 7 days
- **Bid Adjustment:** +50%

### Frequency Cap

- **Impression Cap:** 5 impressions per user per day
- **Weekly Cap:** 20 impressions per user per week
- **Monthly Cap:** 60 impressions per user per month

### Content Exclusions

- Exclude: Parked domains, error pages, sexually suggestive content, sensational/shocking content, sensitive social issues
- Exclude: Mobile app interstitials (low quality placements)
- Exclude: Below-the-fold placements (where possible)
